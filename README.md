# EarnExx + FairScale Prototype

## Overview
EarnExx is a task-based rewards platform on Solana. This prototype demonstrates **FairScore integration using FairScale**, allowing rewards to scale based on wallet reputation.

## Features
- Fetch FairScore for any wallet
- Adjust rewards based on reputation tiers
- Simple frontend demo to visualize multipliers
- Architecture diagram included

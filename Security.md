
# Security Policy

## Reporting a Vulnerability
If you discover a security issue in EarnExx, please contact the team responsibly:

**Email:** security@earnexx.io

## Supported Versions
- Backend: Node.js
- Frontend: Modern browsers
- Smart contracts (future): Solana/Anchor

## How We Handle Reports
1. Confirm receipt of your report within 24 hours.
2. Investigate the issue and assign a severity.
3. Notify the reporter once resolved.
4. Public disclosure is coordinated with the reporter.

## Responsible Disclosure
Please avoid public disclosure until the issue is resolved. Do not exploit the vulnerability for personal gain.
